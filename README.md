## ğŸ§ª Metodologi

### Dataset (3000 Responden)

8 dimensi dengan 36+ fitur:

1. **Kepuasan Hubungan** - Komunikasi, keintiman, konflik
2. **Profil Psikologis** - Gaya kelekatan, kontrol diri, impulsif
3. **Akses & Kesempatan** - DM lawan jenis, komunikasi rahasia
4. **Kedekatan Emosional Lain** - Curhat ke luar, emotional affair
5. **Nilai & Sikap** - Pandangan moral, religiusitas
6. **Komitmen Hubungan** - Status, durasi, rencana masa depan
7. **Pemicu Stres** - Stres kerja, ekonomi, alkohol
8. **Demografi** - Usia, gender, pendidikan, domisili

### Model & Evaluasi

- **Algorithm**: XGBoost Classifier
- **Labeling**: Rule-based multi-class (3 fase)
- **Feature Engineering**: 7 fitur turunan (skor ketidakpuasan, kesempatan, emotional infidelity, dll)
- **Explainability**: SHAP untuk interpretasi model
- **Validation**: Train-test split, confusion matrix, classification report
